# obsisidian в качестве редактора
для просмотра и редактирования используйте программу Obsidian, открывая папку docs как vault

# рекомендации по оформлению текста

## как делать ссылки
ссылки в формате 
```
[понятие в тексте](<как назван файл.md>)
```
например
```
[список вопросов](<2025-осень - список вопросов к экзамену>)
```

## как добавлять примеры кода
для вставки красивых фрагментов кода
```cpp
int main(int argc, char* argv[])
```
используйте блоки между \`\`\` с указанием языка ```cpp```

## как делать заметки
для переноса строк без начала нового абзаца ставьте два пробела в конце строки
